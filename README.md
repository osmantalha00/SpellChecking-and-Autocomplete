# 🔤 Spell Checking and Autocomplete

A Java-based application that performs **spell checking** and **autocomplete** using a **Binary Search Tree (BST)**.

## Features

* Spell checking
* Word suggestion
* Autocomplete
* Java Swing GUI
* Binary Search Tree based structure

## Technologies

* Java
* Swing
* Maven
